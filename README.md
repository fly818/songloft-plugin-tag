# 标签刮削插件

自动修正歌曲标题、艺术家、专辑，下载封面和歌词写入音频文件。支持声纹识别 + 国内源并发搜索，多级目录浏览。

## 安装

1. 下载 [tag.jsplugin.zip](../../releases/latest)
2. 放入 Songloft 的 `data/jsplugins/` 目录
3. 重启 Songloft

## 功能概览

- **智能刮削**：声纹匹配（推荐）或文本搜索，自动选择最佳结果写入标签
- **多源并发**：同时查询多个国内音源，按相似度评分选取最优结果
- **封面下载**：自动从搜索结果中获取封面并写入音频文件
- **歌词下载**：匹配成功后自动下载歌词写入
- **繁简转换**：自动将繁体标签转简体，提高国内音源匹配率
- **批量操作**：选中多首歌曲一键批量刮削，后台异步执行
- **编辑弹窗**：双击歌曲可手动编辑标签、上传封面
- **失败重试**：刮削失败的歌曲自动归档，支持手动输入关键词重试
- **移动端适配**：手机端自动切换单列布局 + 汉堡菜单

## 使用

### 歌曲 Tab — 刮削标签

- **左窗格**：目录树，按 `file_path` 真实目录结构递归展开，支持任意层级深度。点击目录节点筛选歌曲
- **右窗格**：当前目录下的歌曲列表，显示标题/艺术家/专辑/时长/格式
- **状态标记**：已刮削（金色）、失败（红色）、待处理（灰色）
- **封面图标**：有封面的歌曲显示专辑图标，无封面的显示音符图标

工具栏按钮：`[全选] [清空] │ [刮削] [强制] │ [下载封面] [清除封面] │ [停止] [清空记录]`

- **刮削**：选 1 首直接刮削+写入；选多首自动后台批量模式，多源并发搜索
- **强制**：跳过已刮削记录，全部重刮
- **下载封面**：批量下载封面到本地（已有封面的跳过）
- **清除封面**：批量清除嵌入封面（需主程序 v2.7.1+ 支持）

### 失败 Tab — 手动重试

刮削失败的歌曲出现在这里，支持刷新、清空、手动输入关键词刮削。

### 编辑弹窗

双击歌曲卡片打开编辑弹窗，可修改标题、艺术家、专辑、流派、歌词，点击封面区域可上传新封面。

### 配置

在 Header 右侧齿轮按钮打开设置，配置 AcoustID Key 和各音源 API URL。

## 刮削策略

### 声纹匹配（推荐）

1. 主程序需 v2.6.3+，扫描时自动生成 Chromaprint 指纹
2. 在 AcoustID Key 输入框中填入 Key（[免费获取](https://acoustid.org/webservice)）
3. 卡片右上角圆点绿色表示 Key 有效
4. 声纹匹配成功后，自动从国内源富化封面和歌词

> ⚠️ v2.7.0 存在指纹数据污染问题（[songloft#146](https://github.com/songloft-org/songloft/issues/146)），插件会自动降级到文本搜索。

### 文本搜索

声纹不可用或匹配失败时，自动降级为文本搜索：

1. 从文件名/已有标签提取艺术家和标题作为关键词
2. 繁体自动转简体
3. 并发查询所有已启用的国内源
4. 使用 Ratcliff/Obershelp 算法计算相似度评分（0.4×艺术家 + 0.6×标题）
5. 各源有独立权重系数，综合评分 ≥ 0.7 才算匹配成功
6. 选取最高分结果写入标签

## 启用音源

### 国内音源（需配置 API URL）

在「刮削源」区域填入 API URL 后保存，系统自动检测连通性（绿色=可达，红色=不可达）。清空 URL 再保存会自动关闭该音源。

支持多个国内音源，其中部分需用户自行部署 API 服务并填入 URL，部分内置直连无需配置。具体地址请自行获取，插件不内置任何 API URL。

## 支持的音乐格式

| 格式 | 声纹识别 | 标签写入 |
|---|---|---|
| FLAC | ✅ | ✅ |
| MP3 | ✅ | ✅ |
| MP4 / M4A | ✅ | ❌ |
| OGG | ✅ | ❌ |
| WAV | ✅ | ✅ |
| APE | ✅ | ✅ |
| WMA | ❌ | ❌ |

## 技术细节

- **评分算法**：Ratcliff/Obershelp 文本相似度，纯 JS 实现，无外部依赖
- **重试机制**：网络请求自动指数退避重试（最多 2 次）
- **SSRF 防护**：配置 URL 自动拦截内网地址（localhost、127.x、10.x、172.16-31.x、192.168.x）
- **批量异步**：大批量刮削在后台执行，前端轮询进度，不阻塞界面
- **繁简转换**：内置繁体→简体映射表，提高搜索匹配率
