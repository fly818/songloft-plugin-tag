# 标签刮削插件

自动修正歌曲标题、艺术家、专辑，下载封面和歌词写入音频文件。

## 安装

1. 下载 [tag.jsplugin.zip](../../releases/latest)
2. 放入 Songloft 的 `data/jsplugins/` 目录
3. 重启 Songloft

## 使用

插件页面有 3 个标签页：

### 歌曲 — 刮削标签

1. 勾选你要处理的歌曲（支持全选/搜索）
2. 点击 **批量刮削** 开始自动匹配标签
3. 进度条显示实时进度，可随时点 **停止**
4. 关闭页面不影响，重新打开会自动恢复进度

### 失败 — 手动重试

刮削失败的歌曲会出现在这里。输入正确的搜索关键词点 **搜索** 重试，匹配成功后点 **write** 写入标签。

### 编辑 — 手动编辑

搜索歌曲，点进去可以手动修改标题、艺术家、专辑、流派、歌词。支持刮削按钮重新匹配当前歌曲。

## 声纹匹配（可选，强烈推荐）

安装 fpcalc 后，插件会先用声纹精确识别歌曲（AcoustID + MusicBrainz），命中后自动到国内音乐平台拉取封面和歌词。

**Docker：** 进入容器执行 `apk add chromaprint`，然后在插件页面点安装即可。

**其他系统：** 在插件页面点 **安装 fpcalc** 自动下载。

## 环境要求

- Songloft v2.5.0+
