# 标签刮削插件 v2.1.1

自动修正歌曲标题、艺术家、专辑，下载封面和歌词写入音频文件。支持声纹识别 + 国内源并发搜索，多级目录浏览，目录整理与重命名。

## 安装

1. 下载 [tag.jsplugin.zip](../../releases/latest)
2. 放入 Songloft 的 `data/jsplugins/` 目录
3. 重启 Songloft

## 功能概览

### 歌曲 Tab
- **智能刮削**：声纹匹配（推荐）或文本搜索，自动选择最佳结果写入标签
- **多源并发**：同时查询多个国内音源，按相似度评分选取最优结果
- **封面下载**：自动从搜索结果中获取封面并写入音频文件
- **歌词下载**：匹配成功后自动下载歌词写入（支持多源兜底 + 双语合并）
- **繁简转换**：自动将繁体标签转简体，提高国内音源匹配率
- **批量操作**：选中多首歌曲一键批量刮削，支持批量采纳/忽略
- **一键撤销**：自动写回后可恢复原始标签
- **编辑弹窗**：双击歌曲可手动编辑标签（标题/艺术家/专辑/流派/年份/音轨号/歌词）、上传封面
- **元数据健康度**：查看全库标签完整度
- **移动端适配**：手机端自动切换单列布局 + 汉堡菜单

### 校对 Tab
- **交通灯分流**：绿灯（已应用）/ 黄灯（待校对）/ 红灯（失败）
- **Diff 对比面板**：左右对比当前标签 vs 系统推荐，显示标题/艺术家/专辑/流派/年份/音轨号/封面差异
- **多候选切换**：横向滚动卡片，点击切换搜索结果
- **封面画廊**：多源封面搜索，选择后写入
- **批量操作**：勾选多首歌曲，批量采纳或忽略

### 整理 Tab
- **目录整理**：可视化多级目录构建器，支持「+」「-」增删目录层级
- **重命名**：独立的文件重命名功能，支持多字段组合
- **下拉构建器**：目录/文件模板使用下拉选择器（艺术家/专辑/年份/流派/音轨号/标题）
- **实时预览**：选择模板后即时显示目标路径预览
- **预览/执行/撤销**：操作前预览变更，支持一键撤销

## 使用

### 歌曲 Tab — 刮削标签

- **左窗格**：目录树，按 `file_path` 真实目录结构递归展开，支持任意层级深度。点击目录节点筛选歌曲
- **右窗格**：当前目录下的歌曲列表，显示标题/艺术家/专辑/时长/格式
- **状态标记**：已刮削（金色）、失败（红色）、待处理（灰色）
- **元数据健康度**：点击统计栏展开，查看各字段完整度

工具栏按钮：`[全选] [清空] │ [刮削] [强制] │ [下载封面] [清除封面] │ [停止] [清空记录]`

### 校对 Tab — 校对工作台

#### 已应用（绿灯区）
- 显示已自动写回的歌曲列表
- 点击查看详情（标题/艺术家/专辑/流派/年份/音轨号/封面）
- 支持一键撤销，恢复原始标签

#### 待校对（黄灯区）
- 显示评分 0.60~0.85 的歌曲，需人工确认
- Diff 面板：左右对比当前标签 vs 系统推荐，显示所有字段差异
- 候选切换 + 封面画廊 + 批量采纳/忽略

#### 失败（红灯区）
- 显示刮削失败的歌曲和失败原因，支持手动重试

### 整理 Tab — 目录整理与重命名

#### 目录整理模式
- 可视化构建目录结构：点击「+」添加层级，「-」删除层级
- 每级可选择：艺术家 / 专辑 / 年份 / 流派
- 实时预览目标路径

#### 重命名模式
- 可视化构建文件名：点击「+」添加字段，「-」删除字段
- 可选字段：音轨号 / 标题 / 艺术家 / 专辑 / 年份
- 字段间自动用「 - 」连接

#### 操作流程
1. 在目录树中选择目标目录
2. 选择目录整理或重命名模式
3. 配置模板
4. 点击「预览」查看变更
5. 点击「执行」应用变更
6. 支持「撤销」恢复

### 配置

在 Header 右侧齿轮按钮打开设置，配置 AcoustID Key 和各音源 API URL。

#### 高级设置
- **评分阈值**：调整匹配成功的最低分数（默认 0.70）
- **标题/艺术家权重**：调整评分中标题和艺术家的权重比例
- **缓存管理**：查看缓存条目数，清除全部缓存

## 刮削策略

### 声纹匹配（推荐）

1. 主程序需 v2.6.3+，扫描时自动生成 Chromaprint 指纹
2. 在 AcoustID Key 输入框中填入 Key（[免费获取](https://acoustid.org/webservice)）
3. 声纹匹配成功后，自动从国内源富化封面和歌词

### 文本搜索

声纹不可用或匹配失败时，自动降级为文本搜索：

1. 从文件名/已有标签提取艺术家和标题作为关键词
2. 支持「歌手 - 歌名」和「歌名 - 歌手」两种格式
3. 繁体自动转简体
4. 并发查询所有已启用的国内源
5. Ratcliff/Obershelp 算法评分（0.4×艺术家 + 0.6×标题）
6. 综合评分 ≥ 阈值（默认 0.7）才算匹配成功

### 歌词双语合并

- 网易云歌词自动合并原文 + 翻译
- 按时间戳对齐，原文行后紧跟翻译行

### 智能缓存与保护

- 刮削结果本地缓存 24 小时
- 令牌桶限流保护 API（默认 2 req/s）
- 熔断保护：连续失败 10 次后自动熔断 60 秒

## API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/songs` | GET | 歌曲列表（支持搜索） |
| `/scrape/batch` | POST | 批量刮削（异步） |
| `/scrape/incremental` | POST | 增量扫描（自动扫描未处理歌曲） |
| `/scrape/preview/:id` | GET | 预览搜索结果 |
| `/scrape/manual/:id` | POST | 手动刮削 |
| `/organize/preview` | POST | 目录整理预览 |
| `/organize/execute` | POST | 目录整理执行 |
| `/config` | GET/PUT | 配置读写 |
| `/circuit-breaker/status` | GET | 熔断器状态 |
| `/circuit-breaker/reset` | POST | 重置熔断器 |

## 技术细节

- **评分算法**：Ratcliff/Obershelp 文本相似度 + 时长惩罚
- **重试机制**：网络请求自动指数退避重试（最多 2 次）
- **SSRF 防护**：配置 URL 自动拦截内网地址
- **批量异步**：大批量刮削在后台执行，前端轮询进度
- **繁简转换**：内置繁体→简体映射表
- **熔断保护**：连续失败 10 次后自动熔断 60 秒
- **歌词双语合并**：网易云原文+翻译按时间戳对齐
- **增量扫描**：POST /scrape/incremental 自动扫描所有未处理歌曲
