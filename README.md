# 标签刮削插件 v1.0.5

自动修正歌曲标题、艺术家、专辑，下载封面和歌词写入音频文件。

## 安装

1. 下载 [tag.jsplugin.zip](../../releases/latest)
2. 放入 Songloft 的 `data/jsplugins/` 目录
3. 重启 Songloft

## 使用

插件页面有 3 个标签页：

### 歌曲 — 刮削标签

工具栏按钮说明（从左到右）：

| 分组 | 按钮 | 功能 |
|------|------|------|
| 选择 | 全选 / 清空 / 清空记录 | 管理选择状态和刮削记录 |
| 批量 | 下载封面 / 刮削 / 强制 | 下载封面或刮削选中歌曲 |
| 单曲 | 停止 / 预览 | 停止批量任务或预览匹配结果 |

- **刮削**：选 1 首直接刮削+写入；选多首自动切换为后台批量模式，可随时点 **停止**
- **强制**：跳过已刮削记录，对选中歌曲全部重新刮削
- **下载封面**：为选中歌曲批量下载封面到本地（已有封面的跳过）
- **清空记录**：清除刮削标记，允许重新刮削所有歌曲
- 关闭页面不影响批量任务，重新打开会自动恢复进度

### 失败 — 手动重试

刮削失败的歌曲会出现在这里。输入正确的搜索关键词点 **搜索** 重试，匹配成功后点 **write** 写入标签。

### 编辑 — 手动编辑

搜索歌曲，点进去可以手动修改标题、艺术家、专辑、流派、歌词。支持刮削按钮重新匹配当前歌曲。

## 启用声纹匹配（推荐）

声纹匹配通过音频指纹精确识别歌曲，大幅提高刮削准确率。

1. 主程序需 v2.6.3+ 并启用 Chromaprint 指纹计算（扫描时自动生成）
2. 在 AcoustID Key 输入框中填入你的 Key（[免费获取](https://acoustid.org/webservice)）
3. 点击 **保存配置**，卡片右上角圆点亮绿色表示 Key 有效
4. 无需额外安装 fpcalc — 主程序已内置 Chromaprint 支持

## 开启国内音源

国内音源用于拉取封面和歌词，同时作为声纹匹配的补充。

1. 在**刮削源**区域，填入对应的 **API URL**
2. 点 **保存配置**，系统自动检测连通性
3. 卡片右上角圆点：绿色=已配置且可达，红色=不可达，灰色=未配置
4. 清空 URL 再保存会自动关闭该音源

## 支持的音乐格式

| 格式 | 声纹识别 | 标签写入 |
|---|---|---|
| FLAC | ✅ | ✅ |
| MP3 | ✅ | ✅ |
| MP4 / M4A | ✅ | ❌ |
| OGG | ✅ | ❌ |
| WAV | ✅ | ✅ |
| APE | ✅ | ✅ |
| WMA | ❌ | ❌ |

> ✅ APE/WAV 标签写入需要 Songloft v2.6.1+。旧版本写入会静默跳过。
> ⚠️ MP4/M4A/OGG 写入暂未实现，标签只能读取无法回写。
