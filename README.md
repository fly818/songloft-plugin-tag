# 标签刮削插件

自动修正歌曲标题、艺术家、专辑，下载封面和歌词写入音频文件。

## 安装

1. 下载 [tag.jsplugin.zip](../../releases/latest)
2. 放入 Songloft 的 `data/jsplugins/` 目录
3. 重启 Songloft

## 使用

插件页面有 3 个标签页：

### 歌曲 — 刮削标签

1. 勾选你要处理的歌曲（支持全选/搜索）
2. 点击 **批量刮削** 开始自动匹配标签
3. 进度条显示实时进度，可随时点 **停止**
4. 关闭页面不影响，重新打开会自动恢复进度

### 失败 — 手动重试

刮削失败的歌曲会出现在这里。输入正确的搜索关键词点 **搜索** 重试，匹配成功后点 **write** 写入标签。

### 编辑 — 手动编辑

搜索歌曲，点进去可以手动修改标题、艺术家、专辑、流派、歌词。支持刮削按钮重新匹配当前歌曲。

## 启用声纹匹配（推荐）

声纹匹配通过音频指纹精确识别歌曲，大幅提高刮削准确率。

1. 在 AcoustID Key 输入框中填入你的 Key（去 [acoustid.org/webservice](https://acoustid.org/webservice) 免费注册获取永久 Key）
2. 点击 **保存配置**，卡片右上角圆点亮绿色表示 Key 有效
3. fpcalc 由 Songloft 镜像预装，无需额外安装（Docker 环境下）
4. 非 Docker 环境：在插件页面点 **安装 fpcalc** 自动下载对应平台的 fpcalc

## 开启国内音源

国内音源用于拉取封面和歌词，同时作为声纹匹配的补充。

1. 在**刮削源**区域，填入对应的 **API URL**
2. 点 **保存配置**，系统自动检测连通性
3. 卡片右上角圆点：绿色=已配置且可达，红色=不可达，灰色=未配置
4. 清空 URL 再保存会自动关闭该音源

## 支持的音乐格式

| 格式 | 声纹识别 | 标签写入 |
|---|---|---|
| FLAC | ✅ | ✅ |
| MP3 | ✅ | ✅ |
| MP4 / M4A | ✅ | ✅ |
| OGG | ✅ | ✅ |
| WAV | ✅ | ✅ |
| APE | ✅ | ✅ |
| WMA | ❌ | ❌ |

> ✅ APE/WAV 标签写入需要 Songloft v2.6.1+。旧版本写入会静默跳过。

## 环境要求
