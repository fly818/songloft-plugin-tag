# 标签刮削插件 v1.0.7

自动修正歌曲标题、艺术家、专辑，下载封面和歌词写入音频文件。支持多源并发搜索（国内源），多级目录浏览。

## 安装

1. 下载 [tag.jsplugin.zip](../../releases/latest)
2. 放入 Songloft 的 `data/jsplugins/` 目录
3. 重启 Songloft

## 使用

### 歌曲 Tab — 刮削标签

- **左窗格**：目录树，按 `file_path` 真实目录结构递归展开，支持任意层级深度。点击目录节点筛选歌曲
- **面包屑导航**：显示当前路径 `音乐 › 陈小春`，点击任意层回退
- **右窗格**：当前目录下的歌曲列表，显示标题/艺术家/专辑/时长/格式

工具栏按钮：`[全部歌曲] [全选] [清空] [清空记录] │ [下载封面] [清除封面] │ [刮削] [强制] [停止] [预览] [📖说明]`

- **刮削**：选 1 首直接刮削+写入；选多首自动后台批量模式，多源并发搜索
- **强制**：跳过已刮削记录，全部重刮
- **下载封面**：批量下载封面到本地（已有封面的跳过）
- **清除封面**：批量清除嵌入封面（需主程序 v2.7.1+ 支持）
- **全部歌曲**：一键显示所有歌曲文件
- **📖 说明**：弹窗查看插件文档和更新日志

### 失败 Tab — 手动重试

刮削失败的歌曲出现在这里。输入关键词点搜索重试，匹配后点 write 写入。

### 编辑 Tab — 手动编辑

搜索歌曲，点进去修改标题、艺术家、专辑、流派、歌词。支持刮削按钮重新匹配。

## 启用音源

### 国内音源（需配置 API URL）

在「刮削源」区域填入 API URL 后保存，系统自动检测连通性（绿色=可达，红色=不可达）。

| 音源 | 示例 URL |
|------|------|
| 网易云 | `https://music.163.com/api/cloudsearch/pc` |
| QQ音乐 | `https://c.y.qq.com/soso/fcgi-bin/search_for_qq_cp` |
| 酷狗 | `https://songsearch.kugou.com/song_search_v2` |
| 酷我 | `https://kuwo.cn` |

清空 URL 再保存会自动关闭该音源。

### 声纹匹配（推荐）

1. 主程序需 v2.6.3+，扫描时自动生成 Chromaprint 指纹
2. 在 AcoustID Key 输入框中填入 Key（[免费获取](https://acoustid.org/webservice)）
3. 卡片右上角圆点绿色表示 Key 有效

> ⚠️ v2.7.0 存在指纹数据污染问题（[songloft#146](https://github.com/songloft-org/songloft/issues/146)），插件会自动降级到文本搜索。

## 支持的音乐格式

| 格式 | 声纹识别 | 标签写入 |
|---|---|---|

| OGG | ✅ | ❌ |
| WAV | ✅ | ✅ |
| APE | ✅ | ✅ |
| WMA | ❌ | ❌ |
